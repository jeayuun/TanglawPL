Why we created TANGLAW

We have problems in

existing programming languages

Steep Learning Curve

Requires significant time and effort to learn complex syntax

Demands understanding of abstract programming paradigms

Hinders accessibility for new programmers

Limited Support for Geometric Calculations

Manual implementation of basic geometric operations slows development

Causes redundant effort in coding and problem-solving

Increases the likelihood of errors in geometric calculations

Unit Conversion Overhead

Adds extra lines of code and manual checks, increasing complexity

Requires error-prone conversion routines, leading to potential bugs

Decreases productivity and development efficiency

About

Elevate Your Workflow

with TANGLAW

A general-purpose, imperative programming language designed for simplicity and accessibility. Named after "tanglaw", Filipino for light and guidance. Inspired by torchbearers, embodying clarity, knowledge, and the pursuit of understanding.

Guiding Light for Beginners

Promotes inclusivity by making the programming journey clear, straightforward, and accessible.

Ignite Confidence and Exploration

Cultivates confidence and creativity in programming.

Bridge Industry Boundaries

Unites different domains through an easy programming language to learn.
